# Projeto-de-Analise-de-Dados-
Objetivo
Analisar dados de abordagens policiais para identificar padrões relacionados a infrações, prisões, presença de drogas e perfil dos abordados.

Ferramentas utilizadas

-Python

-pandas

-Power BI


Principais análises

-principais motivos das abordagens

-infrações que mais levam à prisão

-porcentagem de abordados com drogas

-distribuição de idade

-comparação entre gêneros

-tempo médio das abordagens
